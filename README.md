# Google-Play-Store-Apps-Analysis
Exploratory Data Analysis of Google Play Store Apps using Python, Pandas, Seaborn, and Matplotlib

## Project Overview
This project analyzes the Google Play Store dataset to understand trends in app ratings, installs, categories, and user reviews.

## Tools Used
Python  
Pandas  
Seaborn  
Matplotlib  
Google Colab  

## Analysis Performed
• Data cleaning and preprocessing  
• Rating distribution analysis  
• Category popularity analysis  
• Relationship between installs and ratings  
• Top installed apps  
• Genre analysis  
• Sentiment analysis of user reviews  

## Key Insights
• Most apps have ratings between 4.0 and 4.5  
• Categories such as Family and Game dominate the Play Store  
• Apps with more installs usually have more reviews  
• Most user reviews are positive  

## Dataset
Google Play Store Apps Dataset
